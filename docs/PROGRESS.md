## Phase 3 実機確認手順（GitHub Pages — 開発機不要）

1. マージ → Actions「Deploy to GitHub Pages」が緑
2. Pixel で `https://kyoujyu88.github.io/VoxelWorld/`（末尾スラッシュ必須、1回リロード推奨）
3. 「AR + 深度センシングを開始」→ 端末をゆっくり動かして部屋を見回す
4. 見た床・壁・物の位置に**ボクセル（立方体）が積もる**か確認:
   - 位置・向き・大きさが実物とおおむね合っているか（**R6**）
   - もし上下や前後が反転していたら、HUD の「**↕ 上下反転**」を押して合うか試す（合った方の flipY 状態も教えてください）
   - 「クリア」でやり直し、「一時停止」で蓄積停止
5. その様子（できればスクショ）と、上下反転は ON/OFF どちらが正しかったかを返してほしい。

> 開発機がある場合は `docs/ENVIRONMENT.md` の方法 B（`adb reverse` + `npm run dev`）でも確認できる。
