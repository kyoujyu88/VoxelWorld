## Phase 3 実機確認手順（GitHub Pages — 開発機不要）

1. マージ → Actions 緑 → Pixel で `https://kyoujyu88.github.io/VoxelWorld/`（末尾スラッシュ、1回リロード）
2. 「開始」→ 部屋をゆっくり見回す。小さめの立方体になり、隙間からカメラ映像が透ける想定
3. 見てほしい:
   - ボクセルが実際の面（床/壁/物）に乗っているか（浮き / めり込み / 位置ズレは？）
   - 「↕ 上下反転」を押すと明確に崩れる/直るか（既定 ON が正しい想定。OFF で崩れれば ON 確定）
   - しばらく見回して**二重壁（ドリフト）**がどの程度出るか
   - 「クリア」でリセット、「一時停止」で蓄積停止
4. スクショ＋印象（整列してる？ノイズ多い？）を返してほしい

> 開発機がある場合は `docs/ENVIRONMENT.md` の方法 B（`adb reverse` + `npm run dev`）でも確認できる。
