## Phase 2 実機確認手順（GitHub Pages — 開発機不要）

1. この変更を `main` にマージ（Actions が自動でビルド＆デプロイ）
2. Actions タブ →「Deploy to GitHub Pages」が緑になるまで待つ
3. Pixel 9a の Chrome で **`https://kyoujyu88.github.io/VoxelWorld/`**（末尾スラッシュ必須）を開く
4. 「AR + 深度センシングを開始」→ 権限許可
5. 端末をゆっくり動かし、下部の**白黒ヒートマップ**を確認:
   - 近い物ほど**明るく**、遠いほど暗く、欠損は透明
   - 統計行の「有効率」「距離 min/中央/max」が動く
6. その様子（スクショか一言）を返してほしい（**R5**）。真っ暗/真っ白/更新されない等があればそれも。

> 開発機がある場合は `docs/ENVIRONMENT.md` の方法 B（`adb reverse` + `npm run dev`）でも確認できる。
